Prerequisites

  • GitHub account
  • GitHub Desktop installed
  • A text editor (Sublime text, BBEdit, or Atom)
  • Atom.io is recommended

Some definitions

Repository
A collection of files
Fork
A copy of a repository
Commit
Save a set of changes in a repository
Merge
Combine the changes in one repository with another
Pull Request
Ask for a merge

Git

  • A commandline tool to manage files and changes to files.
  • Designed by programmers
  • Designed for programmers
  • Not user friendly

GitHub

  • Website to host repositories
  • Designed by programmers
  • Designed for programmers
  • Slightly user friendly

GitHub

  • And manage projects
  • Issues like bug reports and feature requests
  • Project management and progress tracking
  • Access controls
  • Repository history
  • Much more

GitHub Desktop

  • User-friendly interface to Git and GitHub
  • Download a forked repository
  • Create commits
  • Push the commits to GitHub

Setup

  • Create a GitHub account
  • Fork the workshop repository
  • Download the forked repository

Workflow

  • Make some changes to your local copy
  • Commit your changes
  • Push your changes to your GitHub
  • Create a pull request
  • Update your repository
