Spreadsheets: Yuck.

Potential Pain Points

  • Data integrity
  • Formatting
  • Dates
  • Sorting
  • Localization

Spreadsheets: YAY!

Spreadsheets are good for some things.

  • Arithmetic
  • Tables
  • Making people sad.

Plan Ahead

  • Determine why you are using spreadsheets.
  • Design them carefully.
  • Document them more carefully.

Sharing the responsibility

  • Google Drive, Drop Box, SFU Vault.
  • Backup before opening the spreadsheet.
  • Have a "talking stick."

Getting Started

  • What will you be feeding into the spreadsheets?
  • What do you expect to get back from the spreadsheets?

Design

  • Name each column
  • Set the format for all date columns
  • Include the format in the column name
    Birth Date (dd-MMM-YYYY)
  • Use the Freeze Columns feature
  • Don't colour code rows

Atoms

  • One cell === one atomic piece of information
  • A date
  • Full name?
  • Location?

Dates

  • Don't fight the spreadsheet.
  • Choose a format. Stick to it.
  • Use four-digit years.
  • Separate column for notes about the dates.
