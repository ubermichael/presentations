Data Loading

  • Islandora XML Solution Pack Batch Ingest!
  • drush --user=admin islandora_simple_xml_batch_preprocess
    --target=/home/vagrant/ballads
    --namespace=ballads
    --parent=islandora:lyrical-ballads
  • drush --user=admin islandora_batch_ingest

Data Loading

  • One Islandora object per XML file
  • Many data streams for each object:
    1. OBJ: XML content as data stream
    2. TN: thumbnail data stream
    3. MODS: Metadata goodness
    4. FACS: one data stream for each facsimile
