Characters

A character is a number.

  • A = 65
  • a = 97
  • A = 0b1000001
  • a = 0b1100001

But it depends on the encoding!

It's easy to check if a letter is capitalised in binary, just check the sixth bit.

ASCII

  • Only seven bits: 127 characters
  • Includes "control characters" like tab, bell, and backspace
  • Latin alphabet and some punctuation
  • Doesn't include accents.
Started in 1960, based in part on telegraph codes.

ISO Latin

Adds one bit to every character, increases the character count by 128

But what gets included?

Everything

Create 15 "code pages" and every document must include information about which code page it uses.

It gets better

Unicode saves the day

  • Single, open, specification
  • Separates code points and character encodings
  • Space for 1,114,112 characters
  • One code point === one character
  • Three different ways to encode the character as a number
  • One way to decode a character.
Not talking about collation, utf-8/16/32, combiners, diacritics, double width, etc.

Characters and Code Points

  • A character is something in an alphabet
  • A code point is a number identifying a letter in unicode

Notation

  • Unicode notation: U+00C1
  • Decimal: 193
  • Hex: 0xC1
  • XML: &xc1; or just type it!

Don't do this!

              and bold and</i> not italic
